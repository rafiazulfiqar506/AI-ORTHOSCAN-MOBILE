## Project Description
AI OrthoScan is a Flutter-based mobile app
that allows doctors to upload orthopedic 
scans (X-Ray/MRI) for AI-powered diagnosis.
The app uses CNN-based deep learning to detect
fractures, bone injuries and joint abnormalities.

## Modules Implemented

### Core Module - Scan Upload & AI Results
- Upload orthopedic scan images (X-Ray/MRI)
- View AI analysis results
- Heatmap visualization
- Confidence scores display

### Minor Module - Scan Annotation
- Annotate scan images with drawing tools
- Add diagnostic labels
- Queue annotated scans for AI retraining

## Tech Stack
- Flutter (Mobile Frontend)
- Python Flask (Backend - Shared with Web)
- MySQL (Database - Shared with Web)
- TensorFlow/CNN (AI Model)
- OpenCV (Image Processing)
